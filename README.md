# Pippi Voice PWA

這是一個基於 AI 的智慧語音輸入工具，專為解決手機輸入效率低下的問題而設計。

## 功能特色
- **PWA 支援**：可安裝於手機桌面，像原生 App 一樣使用。
- **即時辨識**：串接 Gemini 2.0 Live API，提供流暢的語音轉文字體驗。
- **AI 自動修正**：自動識別「說錯了」、「更正」等口語指令並即時修復內容。
- **智慧格式化**：一鍵將雜亂的逐字稿轉化為精確的條列式重點。
- **繁體中文優化**：專為繁體中文使用者設計，支援中英文混打。

## 技術架構
- **前端**：Vanilla JS + CSS (Mobile First)
- **API**：Google Gemini 1.5/2.0
- **部署**：支援 GitHub Pages 快速部署

## 使用說明
1. 開啟網頁後，點擊右上角設定圖示輸入您的 Gemini API Key。
2. 點擊麥克風開始錄音。
3. 說話完畢後，點擊「自動整理」進行內容美化。
